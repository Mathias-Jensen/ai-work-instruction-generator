# Servo Motor Calibration on Pick-and-Place Unit

## Metadata

Document title: Servo motor calibration on pick-and-place unit  
Equipment or system: Pick-and-place unit  
Target user: Automation technician  
Experience level: Experienced  

## Raw notes

Machine stopped after position error on vertical axis. Put machine in maintenance mode. Checked mechanical movement by hand. No obstruction found. Opened servo drive diagnostics. Encoder position looked offset after motor replacement. Homed axis slowly. Adjusted zero position in controller. Saved calibration parameters. Ran manual jog test up and down. Then ran automatic pick-and-place test with empty tray. Movement smooth. Position error cleared.
